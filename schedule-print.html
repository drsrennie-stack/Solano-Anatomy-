5 modules 8 weeks 5 lab exams 8 weekly cases

    How the week runs. Each day lists the class activity first, then the pre-work that supports it. Pre-work is completed the night before: Sunday prepares Monday's session, Monday prepares Tuesday's, and so on. Cases run Monday through Thursday: intro Monday, prep Tuesday, run Wednesday, TBL and case presentation Thursday. Lab exams run during the afternoon lab session on the day shown, so morning lecture continues the normal case rhythm. Upper and lower extremity muscles are covered primarily in lab and do not appear as dedicated lecture days. Completed homework bundles are handed in at the start of each lab exam, before students enter.
